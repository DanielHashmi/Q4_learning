MCQ 4: Claude plugins also work in OpenClaw, is that it, can't we use plugins from anywhere?

1. OpenClaw supports plugins from Codex, Claude, and Cursor ecosystems as native plugins with full feature parity and can install from any source including npm, ClawHub, local paths, and unlimited marketplace sources.

2. OpenClaw supports plugins from Codex, Claude, and Cursor ecosystems as bundles with selective feature mapping to native features like skills, hooks, and MCP tools, and can install from ClawHub, npm, local paths, archives, and marketplaces.

3. OpenClaw only supports Claude plugins as native bundles with complete feature compatibility and unlimited installation sources including any marketplace or registry.

4. OpenClaw supports plugins from any ecosystem as fully native plugins with complete feature parity and unlimited installation sources without any trust boundary restrictions.
